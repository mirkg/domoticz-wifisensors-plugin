# WifiSensors Domoticz Plugin

## Overview

**WifiSensors** is a project to work with multiple devices connected to Arduino and manage them over wifi.
Thit Domoticz plugin allow to communicate from Domoticz to WifiSensors API and automatically create and update devices.

## Run

1. Clone repository into your domoticz plugins folder
```
cd domoticz/plugins
git clone https://github.com/mirkg/domoticz-wifisensors-plugin.git wifisensors
```
2. Restart domoticz
3. Make sure that "Accept new Hardware Devices" is enabled in Domoticz settings
4. Go to "Hardware" page and add new item with type "WifiSensors"
